# Лысый Гусь (The Bald Goose)

Сайт тёмного фэнтези-кафе для любителей настольных ролевых игр. Гости бронируют столик с предзаказом блюд и онлайн-оплатой, приглашают мастера D&D, создают персонажей и хранят их в профиле.

**Демо:** http://5.35.89.185:8080 · **API-документация (Swagger):** http://5.35.89.185:8080/api/docs

## Описание проекта

### Идея

Настольные ролевые игры популярны, но новичку сложно зайти: нужно найти мастера, собраться где-нибудь, разобраться с правилами и листом персонажа. «Лысый Гусь» — сайт атмосферного кафе, которое закрывает обе стороны: можно просто прийти поесть в таверне, а можно забронировать стол под партию в D&D с настоящим мастером и подготовить персонажа прямо на сайте.

**Аудитория:** поклонники фэнтези и настолок, новички в D&D, любители атмосферных мест.

### Роли пользователей

| Роль | Возможности |
|------|-------------|
| **Гость** | Меню с фильтрами, новости, FAQ, галерея персонажей, форма обратной связи |
| **USER** | Всё выше + бронирование с оплатой, свои персонажи, профиль, достижения и бонусы |
| **MASTER** | Пользователь-мастер D&D: назначается на брони как ведущий игровой сессии |
| **ADMIN** | Админ-панель: CRUD по всем сущностям + дашборд со статистикой |

### Функциональность по экранам

- **Главная** — hero-экран, меню с фильтрами, последние новости, FAQ, контакты с формой обратной связи.
- **Бронирование** — многошаговая форма: дата/время/гости → интерактивная карта зала (свободные и занятые столы) → выбор мастера D&D (ваншот/кампания) → предзаказ блюд → подтверждение → оплата через YooKassa. Черновик переживает перезагрузку страницы.
- **Новости** — список новостей, детальная страница по slug.
- **Персонажи** — публичная галерея, конструктор персонажа по правилам D&D (раса, класс, характеристики, заклинания, снаряжение — справочник загружаются с сервера), экспорт листа персонажа в PDF.
- **Профиль** — редактирование данных, смена пароля с подтверждением текущего, история бронирований с деталями заказа, список своих персонажей, достижения и бонусы.
- **Админ-панель** — 9 разделов: новости, брони, блюда, пользователи, столики, FAQ, заявки, персонажи, статистика. Поиск по ключевым словам в списках.
- **Вход / регистрация** — JWT-аутентификация с ротацией refresh-токенов.

### Особенности

- **Онлайн-оплата** — полный платёжный цикл через YooKassa: создание платежа, redirect, webhook + поллинг статуса, автоотмена зависших броней по TTL.
- **Статусная модель брони** — `DRAFT → PENDING_PAYMENT → CONFIRMED / CANCELLED`.
- **Два языка** — RU/EN на уровне интерфейса (i18next) и контента (двуязычные поля в БД).
- **Тёмная/светлая тема** — без «морганий» при начальной загрузке.
- **Достижения** — in-app уведомления о полученных ачивках, бонусы на счёт.
- **Доступность** — aria-атрибуты в формах и модалках.
- **Производительность** — 10 стрнаиц грузятся не сразу (кроме главной и логина), PDF-библиотека грузится только при клике.

## Архитектура

- **Один origin.** Nginx раздаёт SPA и проксирует `/api/` и `/uploads/` на backend — CORS и кросс-доменные cookie в проде не нужны.
- **Backend** — Трёхслойная (трёхуровневая) архитектура Fastify: `route → service → repository` + Zod-схемы на каждый модуль, централизованный обработчик ошибок с кодами, при старте контейнера сам применяет миграции и seed.
- **Frontend** — Feature-Sliced Design (`app → pages → widgets → features → entities → shared`), направление импортов контролируется `eslint-plugin-boundaries`.
- **Аутентификация** — JWT access (15 мин в памяти) + refresh (7 дней, httpOnly cookie) с ротацией, пароли — argon2.
